# Interactive Workshop — Data Analysis with Pandas in Python

This is an **interactive workshop** focused on mastering the essential tools of the `pandas` library:

- Understanding `Series` and `DataFrame` structures
- Creating DataFrames from dictionaries
- Inspecting data with `info()`, `describe()`, `shape`, and `dtypes`
- Loading data from `.csv` and `.xlsx` files
- Selecting columns and filtering rows
- Using `iloc` and `loc` for positional and label-based indexing
- Adding, dropping, and transforming columns
- Sorting and grouping with `sort_values()` and `groupby()`
- Handling missing values with `isnull()`, `dropna()`, and `fillna()`
- Saving processed data back to CSV or Excel

The workshop includes guided examples, live code execution, and cells you can modify and experiment with directly.

### What You'll Do in This Workshop

- Build your first `DataFrame` from scratch
- Load real data from `.csv` and `.xlsx` files using `read_csv()` and `read_excel()`
- Explore and inspect a dataset with pandas' built-in summary methods
- Filter rows by one or multiple conditions
- Select data by position (`iloc`) vs. by label (`loc`)
- Create new columns from existing ones
- Aggregate data by category using `groupby()`
- Detect and handle missing values (`NaN`)
- Export your cleaned dataset to a new file
