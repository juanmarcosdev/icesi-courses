# Interactive Workshop — String Manipulation & Formatting in Python

This is an **interactive workshop** focused on mastering:

- String slicing and stride
- Case manipulation (`lower`, `upper`, `capitalize`)
- Splitting and joining strings
- Stripping whitespace
- Searching and replacing

- f-strings
