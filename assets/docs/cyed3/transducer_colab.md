# Interactive Workshop — Finite State Transducers with Pyformlang


This is an **interactive workshop** focused on building and working with Finite State Transducers (FSTs) using the `pyformlang` library:


- Understanding the difference between FSAs and FSTs
- Creating transducers with `pyformlang.fst.FST`
- Adding states, transitions, and symbol mappings
- Defining input/output alphabets
- Applying a transducer to an input string
- Composing two transducers
- Visualizing transducers with built-in drawing tools
- Checking transducer properties (determinism, completeness)
- Converting between transducer representations
- Saving and loading transducer definitions


The workshop includes guided examples, live code execution, and cells you can modify and experiment with directly.
